# BlackEye
Phishing attack tools powered by ngrok and localhost

# Installation for Linux:
```
git clone https://github.com/baraeerriyachy/blackeye.git
cd blackeye
chmod +x ./setup.sh
./setup.sh
```

# Installation for Android (termux and equivalents):
```
git clone https://github.com/baraeerriyachy/blackeye.git
cd blackeye
chmod +x ./tmxsp.sh
```
# Usage:
```
sudo ./blackeye.sh
```
